# Smart_Walker_Project
O projeto tem por finalidade fazer um andador inteligente utilizando Arduíno para auxiliar famílias de idosos. 

## Materiais Utilizados:
- Arduino Uno R3
- Placa de ensaio pequena
- Tela LCD 16x2
- Sensor de temperatura[TMP36]
- Piezo
- Potenciômetro
- 1 Resistor de 1KΩ
- 1 Resistor de 220Ω
- Botão
- Bateria 9V
